# INF201 – Advanced Programming

## Overview

This course focuses on advanced programming in Python, covering object-oriented design, scientific computing, and the use of modern development tools. It emphasizes writing efficient, reliable, and well-structured code, along with understanding and improving existing programs.

## Topics

* Object-oriented programming in Python
* Scientific computing
* Version control and development environments
* Testing frameworks and documentation tools
* Debugging techniques
* Performance measurement and optimization
* User interface development
